Markers
=======
	Raw Dimension (Before using pattern maker): 512px*512px
	Font: Open Sans 230pt
	Background RGB value: 244,244,244 (Credit: https://github.com/AR-js-org/AR.js/issues/138#issuecomment-655895046)
	Printing Dimension: ~1.7"*1.7"
	Printer: FUJI XEROX Docuprint CM405df	<- Seems to produce semi-glossy surface which reflects the light too much some times.
	
	Pattern files are generated using this URL: https://jeromeetienne.github.io/AR.js/three.js/examples/marker-training/examples/generator.html
	Settings: Pattern Ratio 0.75, Image size 512px, Border color black.
